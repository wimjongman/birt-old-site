BIRT Tutorial

Using Styles

About Styles

We just mentioned that BIRT uses CSS to define the styles for report items. BIRT also allows you to define the properties in a separate style sheet, then apply the style to a report item, just as in HTML. This allows you to create the style once and use it multiple times. If you need to change the look of a report, you can change the style in one place rather then in many different report items. Again, if you've defined your own styles when creating HTML pages, you're well on your way to understanding the BIRT style system.

Release 1.0 Note: Styles in the first BIRT release work only within a single report (though you can copy & paste them across reports.) A later BIRT release will create separate style sheet files that let you share styles across a suite of reports. This is very similar to the external style sheet files in CSS.

Creating a Style

Let's use the style concept to format the report header. Let's make the report header a blue bar with contrasting text. Instead of formatting each label one-by-one, we'll define a style, then apply it.
  • Select the first label in the table Header band.
  • Select Style-->Add Style from the context menu.
  • The Style Properties dialog appears.
  • Switch to the General page.
  • Enter "Header" in the Name field.
  • Switch to the Background page.
  • Click the Background color button and choose the same blue: third from the right in the top row.
  • Switch to the Font page.
  • Set the Font color to white.
  • Set the Font weight to Bold.
  • Click OK.
Notice that the new style has been applied to the label.

Applying a Style

Next, let's apply the style to the other three header labels. (We could have done this in a single step by selecting all four when we defined the style. We're doing it in two steps just to show how it's done.)
  • Select the three labels: click on the first, then use Ctrl+Click to add the other two to the selection.
  • Choose Style-->Apply Style from the context menu.
  • Choose Header from the list of styles that appears.
The other three cells are now also formatted.

Editing a Style

Suppose we wanted to change the style we just created. Here's how we find and change it.
  • Find or open the Outline view.
  • Scroll down to the bottom and find the Styles node.
  • Open the Styles node to reveal our styles.
  • Double-click the style you want to edit.
