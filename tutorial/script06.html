Editing & Testing a Data Set

Edit a Data Set

Like the data source, the data set appears in the Data Explorer. We can use the data set editor to both change and test the data set.

  • Find or open the Data Explorer.
  • Open the Data Sets node.
  • Double-click the Customers data set. The Data Set editor opens.

Data Set Editor Pages

The data set editor provides the following pages:

Query
Displays your SQL SELECT statement. You can edit this statement to add a WHERE clause, adjust the set of selected columns, and more.
Computed Columns
Computed columns are calculated by BIRT and appear to the rest of the report just like any other column. You can use JavaScript expressions, and even call into Java for advanced business logic.
Input Parameters
SQL SELECT statements can include input parameters that pass data into the query. They are marked using the SQL ? symbol in the query. This page lets you give the parameters a name, a default value, and so on. There must be a one-to-one coorespondence between entries on this page and the ? symbols in your query.
Filters
Additional filters applied to your data. Filters restrict the data passed to the report. The WHERE clause is also a filter: one executed by the database. This tab lets you define additional filters using BIRT (JavaScript) scripting.
Output Columns
Shows the columns within the query, and lets you define a column alias for use in the report. Note that this view shows the columns as the report will see them. The Query page shows the columns as selected from the database.
Preview Results
Lets you test your query and see the output. Very handy to make sure that the data is right before you proceed with report design.
For now, we're just going to test our query.

Test a Data Set

With the data set editor open, do the following:

  • Click on the Preview Results item.

The preview page appears. This page shows the columns selected in your query listed across the top. The actual rows in your query appear down the page.

  • Verify that the list of customers appears.
  • Click OK to close the data set editor.
