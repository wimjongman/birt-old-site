Using a Grid

Create a Grid

Let's add a heading to our report. Our report heading will use the classic "left/center/right" layout: a logo on the left, the report title in the center, and the run date on the right.

The easiest way to create such a layout is to use a grid. A grid is like an HTML table, it tells BIRT the desired layout, and BIRT does the work to make it happen. In this case, we want a one-row grid with three columns each 1/3 the width of the table.

  • Drag a Grid item from the palette into your report above your table.

    Release 1.0M1 Note: It is currently difficult to drop the grid in the right place. Watch for a vertical black line to the left of the table. If all else fails, drop it under the table, in the right half of the page, then move it above the table.

  • Delete the last row by clicking the Grid's tab, then on the gray header to the left of the row. Press Delete.
  • Delete the second row the same way.
  • Select the three column headers.

    Release 1.0M1 Note: The current build does not support multiple selection of columns. For now, just set each column individually.

  • Switch to the General tab in the Property Editor. Notice that the width is unset. As in HTML, if the column width is not set, BIRT will size the column to its content.
  • Set the column widths to 33%. Do this by typing "33" in the width field, then selecting "%" from the adjacent pulldown.
  • Select the middle column. In the General page of the Property Editor, set the text alignment to Center.
  • Select the rightmost column. In the General page of the Property Editor, set the text alignment to Right.

Insert an Image

Next, let's add the company logo to the heading. We'll link to an image on the BIRT web site; but you can also embed an image located on your local disk.

  • Drag an Image item from the palette into the left-most cell of your grid.
  • The Image Builder dialog appears.
  • Ensure that the URL radio button is selected.
  • Enter the following URL: http://www.eclipse.org/birt/tutorial/multichip-4.jpg.
  • Click the Preview button to ensure that the URL is correct.
  • Click Insert to add the image to your report.

Insert the Run Date

Next, let's display the current date in the right-most cell in the heading.

  • Drag a Data item from the Palette into the right-most cell of your grid.
  • The expression builder will appear.
  • Display the current date.

    Release 1.0M1 Note: The current date functionality is not yet available. For now, simulate it by typing "Mar. 1, 2005" with the quotes.

  • Click OK.
  • Set the date format. Select the Format page within the Property Editor.
  • Choose "Medium Date" for the date format.

BIRT uses Java formatting (based on Microsoft Excel) for dates, numbers and strings. In addition, BIRT adds a number of specialized, locale-independent date formats described in the ROM Styles Specification.
