BIRT Tutorial

Creating a Report

Create a Project

The first step with Eclipse is to create a new project. Eclipse uses projects to organize your files. For this demo, let's create a project called "My Reports":

  • Choose File→New→Project. The new project dialog appears.
  • Open the "Business Intelligence and Reporting Tools" group and choose "Report Project.
  • Click Next.
  • Type "My Reports" (without the quotes) as the project name.
  • Click Finish.
  • Click OK when asked to switch to the Report Design perspective.

Create a Report

We're now ready to create our first report.

  • Choose File→New→Report. The new report dialog appears.
  • In the tree view, choose the project we just created: My Reports.
  • Enter the following in the file name field: Customers.rptdesign
  • Click Next.

Choose a Template

The next page of the New Report dialog presents a selection of report templates. Before we choose the one we want, let's walk though what's available.

Blank Report
A simple blank report with no content.
Simple Listing
A tabular listing with no grouping.
Grouped Listing
A tabular listing with grouping. Grouping shows repeated values just once. Grouping also allows you to create subtotals.
Chart & Listing
A tabular listing that adds a chart to the top of the report.
Dual Chart & Listing
Combines charts and tables so you can display two sets of related data in a single report.
Side by Side Chart & Listing
A visual variation of the above.
Mailing Lables
Prints mailing labels on special label stock on your printer.
Letter
A form letter in which database data is "mail-merged" into the text. The letter text can be formatted using a subset of HTML.

Release 1.0M1 Note: Only the Blank Report, Simple Listing and Grouped Listing and Letter work in this build.

We're now ready to choose our template:

  • Select the Grouped Listing.
  • Click Finish.
