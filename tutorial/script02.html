BIRT Tutorial

Creating a Report

Create a Project

The first step with Eclipse is to create a new project. Eclipse uses projects to organize your files. For this demo, let's create a project called "My Reports":
  • Choose File-->New-->Project
  • Open the Simple group and choose Project.
  • Type "My Reports" (without the quotes) as the project name.
  • Click Finish.

Create a Report

We're now ready to create our first report.
  • Choose File-->New-->Report
  • The new report dialog appears.
  • In the tree view, choose the project we just created: My Reports.
  • Enter the report name in the File name field: Customers.rptdesign
  • Click Next

Choose a Template

The next page of the New Report dialog presents a wide range of report templates. Before we choose the one we want, let's walk though what's available.
Blank Report
A simple blank report with no content.
Simple Listing
A tabular listing with no grouping.
Grouped Listing
A tabular listing with grouping. Grouping allows you to show repeated values just once, and to create subtotals.
Chart & Listing
Like a simple listing, but adds a chart to the top of the report.
Dual Chart & Listing
Combines charts and reports so you can display two sets of related data in a single report.
Side by Side Chart & Listing
A visual variation on the above.
Mailing Lables
Helps you create mailing labels that print on special stock on your printer.
Letter
A form letter in which database data is "mail-merged" into the text. The letter text can be formatted using a subset of HTML.

Release 1.0M1 Note: Only the Blank Report, Simple Listing and Grouped Listing and Letter work in this build.

We're now ready to choose our template:

  • Select the Grouped Listing
  • Click Finish
