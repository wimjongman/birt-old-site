BIRT Project

Business Intelligence and Reporting Tools

Welcome!

BIRT is an open source, Eclipse-based reporting system that integrates with your application to produce compelling reports for both web and PDF.

BIRT Release 1.0 provides core reporting features such as report layout, data access and scripting. BIRT is currently in the community review phase prior to the 1.0 Release. Please try BIRT and tell us what you think by filling bugs reports & enhancement requests through Bugzilla as explained on the community page.

Download the BIRT 1.0 Release Review slides (.pdf)

New to BIRT?

The topics below help you quickly learn what BIRT is, learn how to use this site and -- most importantly -- try BIRT out for yourself. Just follow the simple steps below. Note that the menu on the left contains each of these items for ease of navigation.

Introduction
Summary of BIRT, what it does, and how it fits into your Eclipse development environment and J2EE server environment.
Sample Gallery
Provides a variety of examples that show BIRT in action.
Download & Install
Download & install BIRT and the software it requires.
Sample Database
BIRT provides a sample database used by the tutorial and some of the example reports. This database lets you learn BIRT on a simple, known database before you experiement with your own, real-world data sources.
Tutorial
Try out BIRT for yourself by following this simple tutorial that walks you through building a report from scratch.
Integration
You can integrate BIRT into your application using the BIRT web app, design API, or engine API.
