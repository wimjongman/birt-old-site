BIRT Stable Build S20050314

New & Noteworthy

Report Designer

  • Implemented the extension system with charts.
  • Added JavaScript native objects corresponding to some of the ROM elements.
  • Added a validation framework to handle checks across multiple properties.
  • Drag & drop will insert a new element at the insert position even if the mouse cursor is in the margin area of the designer.
  • Changed the look and feel of margin border. Made it similar to Word.
  • Made the margin area visible if the element is expanded within the content area.
  • Implemented column and row back ground color.
  • Added edit action to the context menu of label item, text item, etc.

Report Engine

  • FOP customization. Now users can customize FOP through a config file birt_fop.xml. They can ask FOP to use additional font when rendering. This addresses a newsgroup question about supporting Chines characters in PDF output.
  • Handling spaces in textual items. Consecutive space characters in HTML input for text items are trimmed, while consecutive space characters entered as plain text are preserved.
  • Migrated to Java 1.4 logging framework as suggested by a user in the newsgroup. The engine now uses Java 1.4 logging framework.
  • Completed simple master page support, including two new properties footerHeight and headerHeight.
