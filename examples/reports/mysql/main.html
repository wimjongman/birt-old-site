BIRT Report Examples

BIRT MySQL DB Driver Install

Introduction

This example demonstrates the use of BIRT to create reports on data that is stored in a MySQL database.

The user is guided through the steps to create a JDBC data source, specify MySQL driver files to use for the BIRT report designer, the standalone BIRT report viewer, and the BIRT runtime engine when BIRT reports are deployed under Tomcat.
