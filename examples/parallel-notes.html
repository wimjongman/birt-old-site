Parallel Report Example

Notes

Setup

This report uses a (rather old) Microsoft Access database called "emp". To run this report, you'll need to download and install this database:

  • Download the sample database zip file.
  • Unzip the database into a convenient location.
  • Create a System DSN for this database using the Microsoft Access driver. Call it "EmpExample".

About the Report

The report has an outer query that displays information about an employee. Then, it has two "inner" queries that display details for this employee. Because we are showing two sets of details, we need two "subreports", and we use a grid to list them side-by-side.

The report prints for Employee 2. The queries are set up to all get data for the same employee. (A later revision will take the employee ID as a parameter.)
